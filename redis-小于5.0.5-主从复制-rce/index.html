Redis 小于5.0.5 主从复制 RCE

漏洞描述

在2019年7月7日结束的WCTF2019 Final上，LC/BC的成员Pavel Toporkov在分享会上介绍了一种关于redis新版本的RCE利用方式，比起以前的利用方式来说，这种利用方式更为通用，危害也更大

Pavel Toporkov的分享

漏洞影响

1

Redis <= 5.0.5
